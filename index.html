Google Analytics 4 Events and Info

Seach

Submitting the form adds the GA4 built-in event search into the dataLayer array.

The search event requires a built-in parameter called search_term (More information).

On GTM, we need 1 Data Layer Variable variables: search_term (required). We can have other custom parameters like region and we need 1 DLV for that on GTM as well.

In theory, the region parameter is conflicting with GA's built-in dimension with the same name. Nothing happens. Make sure you go to GA > Admin > Custom definitions > Create custom definition

  • Dimension name: Any custom name you want. In this case, we use Region but you can use something like School
  • Scope: Event
  • Description: optional but fill out for future reference
  • Event parameter: Attached to what event. For example, in this case here, we are dealing with the search event so we select search

Sign_up

Submitting the form adds the GA4 built-in event sign_up into the dataLayer array.

Parameters:

This event takes 1 optional built-in parameter called method (More information). For example: Facebook | Google | Apple

On GTM, we need 1 Data Layer Variable variable: method.

Since this is a built-in event with built-in parameter(s), we do NOT need to create any Custom Dimensions.

Login

Parameters:

This event takes 1 optional built-in parameter called method (More information). For example: Facebook | Google | Apple

On GTM, we need 1 Data Layer Variable variable: method.

Since this is a built-in event with built-in parameter(s), we do NOT need to create any Custom Dimensions.

Share

Parameters:

This event takes 3 optional built-in parameters: (More information).

  • method: type string, example: Twitter | Facebook
  • content_type: type string, example: image | blog | discount | coupon_code
  • item_id: type string, example: 1323 | coupon_xed834b

On GTM, we need 3 Data Layer Variable variables: method, content_type, and item_id.

Since this is a built-in event with built-in parameter(s), we do NOT need to create any Custom Dimensions.
